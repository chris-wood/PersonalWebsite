☕ The Daily Grind

I am a Ph.D. student at UCI studying applied cryptography and network security under Gene Tsudik. I am also a core developer of CCNx in the Computer Science Laboratory at PARC (Palo Alto Research Center).

I also try to run or bike, play guitar, write code unrelated to my professional life, take pictures of random things, study foreign languages, and consume far too much coffee on a daily basis.

📓 Research Interests

Content-centric networking architectures and protocols, computer and network security and privacy, cryptographic algorithms and engineering, software engineering, and heterogeneous computing.

💻 Active Projects

  • CCNx: heavy research and development of the CCNx 1.0 software stack
  • ccn-lite: code contributions
  • COOL (COntent Oriented Lisp): an experimental language built on top of CCN
  • iris: a prototypical CCN forwarder written in Rust
  • CBC: a portable and versatile content-based cryptography library for CCN applications written in C
